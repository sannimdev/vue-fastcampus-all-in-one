# vue-04-movie-app

# 1. OMDB API

## 1-1. 가입 절차

1. [omdbapi.com](http://www.omdbapi.com/) 가입
2. 이메일을 통해 키 수신
3. 무료버전 기준 일 1000건 무료

## 1-2. Rest API

- https://www.omdbapi.com/?s=[검색어]&apikey=[키]&page=[페이지]
  - s: 검색어
  - apikey: 키
  - page: 페이지 (10개 단위)

## 1-3. API 사용 with 라이브러리

- axios 사용

  - npm i axios 또는 vue ui를 실행한 뒤 설치 (의존성)
